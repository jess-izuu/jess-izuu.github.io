Data Science Projects



Google/Youtube Recipe Search

Examined Recipe Search Results for 200+ International Dishes input into Google and Youtube. Explored sentiment analysis and examined frequency relationship between search vs. title and description results to build predictive model using tf-idf, Youtube API, SQLite, and Tableau.

Examining the Subjectivity of Personality Tests

Worked with a Kaggle dataset of 1Million+ responses to better understand the Big Five of Psychology Personality Tests. Displayed categorical ranking data in Tableau and analyzed the shape and distrbution of +/- keys in each of the Big Five

Flashcards iOS App

Used XCode and Swift programming to create a simple flashcard mobile application where users can add flashcards, flip when revealing the answer, and slide to go forward/back to the next/previous flashcard.
